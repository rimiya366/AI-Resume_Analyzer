# 📄 Resume & Job Matcher

An AI-powered Resume and Job Description Matcher built using Streamlit and Ollama.

## Features

- Upload Resume (PDF/TXT)
- Upload Job Description (PDF/TXT)
- Local LLM Analysis using Ollama
- ATS-style Match Score
- Strengths
- Missing Skills
- Resume Improvement Suggestions
- Download Report

## Tech Stack

- Python
- Streamlit
- PyMuPDF
- Ollama
- Requests
